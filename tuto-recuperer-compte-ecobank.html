Guide : Comment débloquer et récupérer votre compte Ecobank

Votre application Ecobank Mobile affiche un message d'erreur ou votre compte est bloqué après plusieurs tentatives de connexion ? Pas de panique. En 2026, il existe des méthodes simples pour réinitialiser votre accès sans forcément vous déplacer en agence.

Note de sécurité : ArtMatique et Ecobank ne vous demanderont jamais votre code PIN ou vos identifiants par message. Ne partagez jamais ces informations confidentielles avec des tiers.

Pourquoi votre compte est-il bloqué ?

Le blocage survient généralement pour trois raisons :

  • Trois tentatives de code PIN erronées sur l'application mobile.
  • Une inactivité prolongée du compte (compte dormant).
  • Une mise à jour de vos informations KYC (Know Your Customer) requise par la banque.

Méthode 1 : Utilisation du portail Ecobank Online

Si l'application mobile est bloquée, le portail web est souvent la solution la plus rapide :

  • Rendez-vous sur le site officiel Ecobank Online (Internet Banking).
  • Cliquez sur l'option "Forgot Password / PIN" (Mot de passe oublié).
  • Saisissez votre numéro de compte ou votre adresse e-mail associée.
  • Vous recevrez un code de vérification (OTP) par SMS ou par mail pour créer un nouveau code secret.

Méthode 2 : Récupération via le service Client (WhatsApp & Réseaux)

Ecobank dispose d'un assistant virtuel très efficace pour le déblocage rapide :

  • Rafiki sur WhatsApp : Enregistrez le numéro officiel de l'assistance Ecobank (disponible sur leur site) et envoyez "Bonjour". Suivez les instructions pour le service client.
  • Réseaux Sociaux : Contactez les pages officielles certifiées sur Facebook ou X (Twitter) en message privé avec votre numéro de compte. Les conseillers peuvent souvent déclencher une réinitialisation à distance après vérification de votre identité.

Méthode 3 : L'application Ecobank Mobile (Option "Besoin d'aide")

Si vous avez encore accès à l'écran de connexion :

  • Ouvrez l'application et cliquez sur "Besoin d'aide ?" en bas de l'écran.
  • Sélectionnez "Réinitialiser le code PIN".
  • Il vous sera demandé de répondre aux questions de sécurité que vous avez configurées lors de la création de votre compte.

Conseils pour éviter un futur blocage

Pour ne plus perdre l'accès à vos fonds, nous vous recommandons :

  • D'activer l'authentification biométrique (Empreinte digitale ou Face ID) dans les paramètres de l'application. Cela évite les erreurs de saisie de code PIN.
  • De mettre à jour régulièrement vos coordonnées (numéro de téléphone et email) en agence pour toujours recevoir les codes de récupération (OTP).
